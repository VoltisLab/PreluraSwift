## How to use

1. After adding all settlements for an area, change `- [ ]` to `- [x]` for that row.
2. Run the deduplication script after each batch (or when all are done):  
   `python3 Scripts/dedupe-locations.py`
3. See [docs/location-data.md](location-data.md) for script usage and backend-vs-device decision.
